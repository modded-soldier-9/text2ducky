Command Reference

            

                
                

                    
Basic Commands

                    
    
                        
  • 
                            STRING - Type text
                            
                                Types the text that follows the command
                            
                        
    • 
                        
  • 
                            ENTER - Press Enter key
                            
                                Simulates pressing the Enter key
                            
                        
    • 
                        
  • 
                            DELAY - Add delay (ms)
                            
                                Adds a pause in milliseconds
                            
                        
    • 
                        
  • 
                            REM - Add comment
                            
                                Adds a comment (not executed)
                            
                        
    • 
                    

                

                
                

                    
Special Keys

                    
    
                        
  • 
                            GUI - Windows Key
                            
                                Windows/Super key for system commands
                            
                        
    • 
                        
  • 
                            ALT - Alt Key
                            
                                Use with TAB to switch windows
                            
                        
    • 
                        
  • 
                            SHIFT - Shift Key
                            
                                Used in key combinations
                            
                        
    • 
                        
  • 
                            CTRL - Control Key
                            
                                Common shortcut key (Copy, Paste, etc)
                            
                        
    • 
                    

                

                
                

                    
Common Combinations

                    
    
                        
  • 
                            GUI r - Run dialog
                            
                                Opens Windows Run dialog box
                            
                        
    • 
                        
  • 
                            ALT TAB - Switch window
                            
                                Switch between open windows
                            
                        
    • 
                        
  • 
                            CTRL SHIFT ESC - Task Manager
                            
                                Open Windows Task Manager
                            
                        
    • 
                        
  • 
                            CTRL c - Copy
                            
                                Copy selected text to clipboard
